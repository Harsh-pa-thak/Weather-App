## Tech Stack

- React 19
- Vite 8
- Material UI (MUI) + MUI Icons
- Emotion + styled-components
- Fetch API

## API

- Provider: OpenWeatherMap  
- Endpoint: https://api.openweathermap.org/data/2.5/weather  

### Request Parameters
- `q` (city)
- `appid` (API key)
- `units=metric`

### Data Used
- City name
- Temperature
- Min / Max temperature
- Humidity
- Weather description
